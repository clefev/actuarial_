Independence

class actuarial.copula.Independence[source]

Bases: actuarial.copula.Copula.Copula

The independence copula is a special copula in which each individual component is independent of each other.

Therefore, the marginals \(U_i\) of the independence copula are uniform distributions independent of each other:

\[U_i \perp\!\!\!\perp U_j\]

for all \(i \neq j\).

The independence copula is a special case of the Gaussian copula with a correlation matrix equal to the identity matrix.

cdf(x, keepdims=False)[source]

Evaluate the cumulative distribution function.

Note

The cdf of the independence copula is given by

\[C(u_1, ..., u_d) = \prod_{i=1}^d u_i\]

where \((u_1, ..., u_d) \in [0,1]^d\).

Parameters

  • u (array-like) – Values (between 0 and 1) for which the cdf should be evaluated. The size along the last axis must be equal to the dimensionality of the copula.

  • keepdims (bool, optional) – If True, the last axis which is reduced is left in the result as dimension with size one (otherwise, it will be removed). With this option, the result will broadcast correctly against the input array. The default is False.

Returns

Evaluated cdf. Array with a float dtype and the same shape as u except that the last axis has a size 1 if keepdims is True or is removed if it is False.

Return type

scalar or array

Raises

ValueError – If the size of u along its last axis is not equal to the dimensionality of the copula or if u contains values that are not between 0 and 1.

random(size=None, seed=- 1)[source]

Draw d-dimensional random samples of joint uniform distributions.

Note

The following methodology is used to generate a d-dimensional random sample from the independence copula:

  1. Draw d independent random samples \(u_1, ..., u_d \in (0,1)\) from an uniform distribution.

  2. Return \((u_1, ..., u_d)\).

Parameters

  • size (int or tuple of int or None, optional) – Number of samples to be drawn. Because each sample is d-dimensional, the output shape will have an additional axis with size d in last position. If None, a single d-dimensional sample is returned. The default is None.

  • seed (int or array-like of int or SeedSequence or Generator or None, optional) – Seed to use for the random number generator. The default uses the random number generator associated with the instance.

Returns

Random samples. Array of values between 0 and 1 with the same shape as size except that an additional axis with size d has been added in last position.

Arithmetic

class actuarial.superclass.Arithmetic[source]

Bases: object

Superclass to handle arithmetic operators such as addition, substraction, multiplication, division and power.

All these arithmetic operators are available at once by implementing the __arithmetic__ abstract method in the derived class. This abstract method describes how arithmetic operators (+, -, *, / and **) should behave on the object.

abstract __arithmetic__(value, operator)[source]

Abstract method describing how arithmetic operators (+, -, *, / and **) should behave on the object.

The operator argument is a function representing a generic arithmetic operator.

For example, the derived class could implement the following __arithmetic__ method:

def __arithmetic__(self, value, operator):
    return operator(self.attr, value)

By implementing this single abstract method, all arithmetic operators mentioned above will be available (e.g., object + value will return the result of object.attr + value, object - value will return the result of object.attr - value, …).

Import the Arithmetic superclass.

from actuarial.superclass import Arithmetic

Create a class Example inheriting from the Arithmetic superclass and having an amount attribute representing a number. The application of arithmetic operators on the instance (+, -, *, / and **) should return the result of this arithmetic operator applied on the amount attribute of the instance.

class Example(Arithmetic):

    def __init__(self, amount: float):
        self.amount = float(amount)

    def __arithmetic__(self, value, operator):
        return operator(self.amount, value) # return the result of the arithmetic operator applied on the 'amount' attribute of the instance

If the application of an arithmetic operator on the instance should return the result through a new instance instead, the __arithmetic__ method could be defined as

def __arithmetic__(self, value, operator):
    return Example(amount = operator(self.amount, value)) # return a new instance containing the result of the arithmetic operator applied on the 'amount' attribute of the existing instance

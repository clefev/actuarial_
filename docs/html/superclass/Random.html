Random

class actuarial.superclass.Random[source]

Bases: object

Superclass for building random models.

A random model is a model able to generate randomness. A probability distribution is an example of random model. We can indeed generate random samples from the distribution.

Each Random instance shares the following features:

  • The randomness is generated through a random method.

  • A random number generator (rng) is associated to each instance and accessible through the get_rng method.

  • The rng associated to the instance can be customized through the set_rng method (e.g., to provide a new rng to the instance or create a new one is a custom seed).

propagate_rng

Indicate whether the random number generator used for this instance should be propagated to the other instances in case of nested randomness (i.e. when calling other Random instances to generate randomness). If True, other instances should use the same random number generator as the one associated with this instance. If False, other instances should use their own random number generator.

get_rng(seed=- 1)[source]

Get a random number generator.

Parameters

seed (int or array-like of int or SeedSequence or Generator or None, optional) – Seed to use for the random number generator. If the seed is -1, the random number generator associated with the instance is returned. Otherwise, a new random number generator is created and returned. This new random number generator is built using a default_rng() instance of numpy with the provided seed. The default is -1.

set_rng(seed=None, propagate=None)[source]

Set the random number generator specific to the instance.

Parameters

  • seed (int or array-like of int or SeedSequence or Generator or None, optional) – Seed to use for the random number generator. If seed is -1, no change is applied and the current random number generator is kept. Otherwise, a new random number generator is created and associated with the instance. This new random number generator is built using a default_rng() instance of numpy with the provided seed. The default is None.

  • propagate (bool or None, optional) – Indicate whether the random number generator used for this instance should be propagated to the other instances in case of nested randomness. If None, no change is applied and the current option is kept. The default is None.

Returns

Instance.

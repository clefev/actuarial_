Broadcast

class actuarial.superclass.Broadcast[source]

Bases: object

Superclass implementing broadcasting method to mimic how input aruguments are broadcast with the instance. The superclass also confers properties such as the shape, number of dimensions and size of the instance.

static _asarray(x)[source]

Convert the input argument to a numpy array.

The conversion occurs only if the input argument is not a Broadcast instance and is not None. Otherwise, the original argument is returned as is. In other words, this method is equivalent to the asarray function of the numpy package except that Broadcast instance and None value are not converted.

Parameters

x (array-like or Broadcast or None) – Input to convert to a numpy array.

Returns

Input converted to a numpy array. If the input is a Broadcast instance or is None, the conversion doesn’t occur and the input is returned as is.

Return type

array or Broadcast or None

class actuarial.superclass.Dictionary[source]

Bases: dict

Superclass for building dictionary.

A dictionary is a collection of key-component pairs. The key is represented by a string and the component by an object.

This superclass is based on the base python dictionary (inherits from the dict instance) for which some enhancements have been added:

  • Possibility to accept only one specific type of component within the dictionary (e.g., a dictionary contatining only integers as components).

  • Implementation of an add method to add new components to the dictionary.

  • Implementation of a broadcast method.

  • Improvement of the pop method (now allows to pop several components at once).

  • Iteration over the components of the dictionary instead of the keys (see __iter__).

  • Possibility to get a single component of the dictionary by either its key or its index, and possibility to get multiple components by either a slice or list of keys (see __getitem__).

pop(keys=- 1)[source]

Remove the specified components from the instance. The removed components are returned by the method.

Parameters

keys (str or int or slice or list of str, optional) – Keys of the components to return and remove from the instance. It can be a single component if a string or integer is provided, or multiple components if a slice or list of strings is provided. In the last case, a new instance containing these components will be returned. The default returns and removes the last component.

Returns

Components removed from the instance. If a string or integer is provided, a single component is returned. If a slice or list of strings is provided, multiple components are returned through a new instance.

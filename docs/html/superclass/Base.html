Base

class actuarial.superclass.Base[source]

Bases: object

Superclass implementing basic methods to set new attributes, delete existing attributes, copy the instance and print the instance in a nice format when using the print function.

set_attributes(**kwargs)[source]

Set in-place the attributes of the instance.

Parameters

**kwargs (any) – Attributes to set. The name of the argument is the name of the attribute and the value of the argument is the corresponding value of this attribute. If the name doesn’t exist, a new attribute will be created and if it already exists, its value will be replaced by the provided one.

Returns

Instance.

Return type

Delete in-place the attributes from the instance.

Parameters

*args (str) – Name of attributes to delete. If the name of an attribute does not exist within the instance, it will be skipped.

Returns

Instance.

copy(deep=False)[source]

Return a copy of the instance.

Parameters

deep (bool, optional) – If True, a deep copy of the instance is returned and if False, a shallow copy is returned. A shallow copy creates a new instance but doesn’t create a copy of nested objects while a deep copy does. The default is False (shallow copy).

Returns

Copy of the instance.

Return type

_dict_to_print()[source]

Return a dictionary with the attributes and properties to print when calling the print function on the object.

The dictionary will contain all attributes and properties mentionned in the _print attribute of the class. Therefore, the _print attribute must be a tuple containing the name of all attributes and properties of the instance to print. If a name doesn’t exist as an attribute or property, it will be skipped.

Returns

Dictionary with the attributes and properties of the instance to print.

Return type

dict

__repr__()[source]

Object representation in string format. The representation includes the name of the module where the object comes from, the name of the class and a list of attributes and properties given by the _dict_to_print method.

Parametric

class actuarial.superclass.Parametric[source]

Bases: object

Superclass for building parametric models.

A parametric model is described by a set of parameters. The parameters are given through a Parameters instance. Therefore, each parameter has a name, type and potential constraints.

The normal distribution is an example of parametric model. The parameters are the mean \(\mu\) and the standard deviation \(\sigma\) of the distribution. Both parameters are scalar type. The mean has no constraints while the standard deviation must be strictly positive.

__init__(parameters)[source]
Parameters

parameters (Parameters) – Parameters of the instance. The parameters must be initialized, respect their constraints and be able to broadcast together.

Raises

  • TypeError – If parameters is not a Parameters instance.

  • ValueError – If parameters cannot be broadcast together. If parameters are not valid when using the control method.

abstract property _parametrization

Abstract property giving the parametrization used by the class. The parametrization is given through a Parameters instance where the parameters are not initialized. The parameters attribute of an instance should be an initialized copy of this parametrization.

Exponential

class actuarial.function.Exponential[source]

Bases: actuarial.superclass.Parametric.Parametric, actuarial.function.Function.Function

A shifted and scaled exponential function is a parametric function based on the natural exponential function and having the following form:

\[f(x) = ae^{bx} + c\]

where \(a\) is the scale parameter multiplying the exponential, \(b\) controls the rate of the exponential and \(c\) is a shift translating the exponential.

__init__(scale=1, rate=1, shift=0)[source]
Parameters

  • scale (array-like, optional) – Scale parameter \(a\). This parameter multiplies the exponential. The default is 1.

  • rate (array-like, optional) – Rate parameter \(b\). This parameter controls the rate of the exponential. The default is 1.

  • shift (array-like, optional) – Shift parameter \(c\). This parameter translates the exponential. The default is 0.

Raises

ValueError – If the parameters cannot be broadcast together.

Form

\(f(x) = ae^{bx} + c\)

Parameters
scale = \(a \in \mathbb{R}\)
rate = \(b \in \mathbb{R}\)
shift = \(c \in \mathbb{R}\)

Domain

\(\mathbb{R}\)

Range
\((c, +\infty)\) if \(a > 0\)
\((-\infty, c)\) if \(a < 0\)

Note

\[f(x) = ae^{bx} + c\]

for all \(x \in \mathbb{R}\).

derivative(x, order=1)[source]

Compute the derivative.

Note

The derivative of a (shifted and scaled) exponential function is given by

\[\frac{d^n f(x)}{dx^n} = a b^n e^{bx}\]

for all \(x \in \mathbb{R}\) and \(n \in \mathbb{N}_0\).

Parameters

  • x (array-like) – Values for which the derivative should be evaluated.

  • order (int, optional) – Order of the derivative. The default is 1.

Returns

Evaluated derivative. Array with a float dtype and the same shape as the broadcast of x with the function parameters.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the function parameters. If order is not a positive integer.

integrate(lower, upper)[source]

Compute the definite integral.

Note

The definite integral of a (shifted and scaled) exponential function is given by

\[\int_{x_1}^{x_2} f(x) dx = \frac{a}{b} (e^{bx_2} - e^{bx_1}) + c (x_2 - x_1)\]

for all \(x_1, x_2 \in \mathbb{R}\).

Parameters

  • lower (array-like) – Lower bound of the definite integral interval.

  • upper (array-like) – Upper bound of the definite integral interval.

Returns

Evaluated integral. Array with a float dtype and the same shape as the broadcast of lower and upper with the function parameters.

Return type

scalar or array

Raises

ValueError – If lower and upper cannot be broadcast with the function parameters.

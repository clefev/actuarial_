Piecewise function

class actuarial.function.PiecewiseFunction[source]

Bases: actuarial.function.Function.Function

__init__(intervals, function, right_closed=False, shift=False)[source]
Parameters

  • intervals (array-like) – Partition \(x_0 < x_1, ... < x_n\) dividing the domain of the piecewise function. The domain of the piecewise function will be divided into the following subintervals: \([x_0,x_1)\), \([x_1,x_2)\), …, \([x_{n-1},x_n)\) and \([x_n,\infty)\). The provided values must be strictly increasing.

  • function (Function) – Parametric function to be applied on each interval of the piecewise function. Parameters of the function must be given for each interval. The first interval of the piecewise function [intervals[0], intervals[1]] will use the function parameters from index 0, the second interval [intervals[1], intervals[2]] will use parameters from index 1, … and the last interval [intervals[N], inf] will use parameters from index N.

  • right_closed (bool, optional) – Indicate whether the function \(f_i\) should be used on the right-closed interval \((x_i,x_{i+1}]\) (True) or the left-closed interval \([x_i,x_{i+1})\) (False). The default assumes a left-closed interval (False).

  • shift (bool, optional) – Indicate whether the function \(f_i\) should be horizontally shifted. If True, the function \(f_i\) at point \(x\) will be evaluated as \(f_i(x-x_i)\) instead of \(f_i(x)\). The default assumes no shift (False).

Raises

  • TypeError – If function is not a Function and Parametric instance.

  • ValueError – If intervals is not along the first axis. If function has not the same size as intervals along the partition axis.

evaluate(x)[source]

Evaluate the value of the function.

Note

The value of a piecewise function is given by

\[\begin{split}f(x) = \begin{cases} f_0(x) & \text{if } x < x_1, \\ f_1(x) & \text{if } x_1 \leq x < x_2, \\ ~ \vdots \\ f_{n-1}(x) & \text{if } x_{n-1} \leq x < x_n, \\ f_n(x) & \text{if } x \geq x_n. \end{cases}\end{split}\]

If the right_closed attribute is set to True (and not False), the inequalities above should change: ‘\(<\)’ becomes ‘\(\leq\)’ and ‘\(\leq\)’ becomes ‘\(<\)’.

If the shift attribute is set to True (and not False), the function \(f_i(x)\) above should be replaced by \(f_i(x-x_i)\).

Parameters

x (array-like) – Values for which the function should be evaluated.

Returns

Evaluated function.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the instance.

derivative(x, order=1)[source]

Compute the derivative.

Note

The derivative of a piecewise function is given by

\[\begin{split}f'(x) = \begin{cases} f_0'(x) & \text{if } x < x_1, \\ f_1'(x) & \text{if } x_1 \leq x < x_2, \\ ~ \vdots \\ f_{n-1}'(x) & \text{if } x_{n-1} \leq x < x_n, \\ f_n'(x) & \text{if } x \geq x_n. \end{cases}\end{split}\]

If the right_closed attribute is set to True (and not False), the inequalities above should change: ‘\(<\)’ becomes ‘\(\leq\)’ and ‘\(\leq\)’ becomes ‘\(<\)’.

If the shift attribute is set to True (and not False), the function \(f_i(x)\) above should be replaced by \(f_i(x-x_i)\).

Parameters

  • x (array-like) – Values for which the derivative should be evaluated.

  • order (int, optional) – Order of the derivative. The default is 1.

Returns

Evaluated derivative.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the instance. If order is not a positive integer.

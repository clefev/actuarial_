Linear

class actuarial.function.Linear[source]

Bases: actuarial.superclass.Parametric.Parametric, actuarial.function.Function.Function

A linear function is a function whose graph is a straight line, that is, a polynomial function of degree zero or one:

\[f(x) = ax + b\]

where \(a\) and \(b\) are respectively the slope and intercept of the function.

__init__(slope=1, intercept=0)[source]
Parameters

  • slope (array-like, optional) – Slope parameter \(a\). The default is 1.

  • intercept (array-like, optional) – Intercept parameter \(b\). The value of the function in 0 will be equal to this intercept. The default is 0.

Raises

ValueError – If the parameters cannot be broadcast together.

evaluate(x)[source]

Evaluate the value of the function.

Note

The value of a linear function is given by

\[f(x) = ax + b\]

for all \(x \in \mathbb{R}\).

Parameters

x (array-like) – Values for which the function should be evaluated.

Returns

Evaluated function. Array with a float dtype and the same shape as the broadcast of x with the function parameters.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the function parameters.

derivative(x, order=1)[source]

Compute the derivative.

Note

The derivative of a linear function is constant and equal to its slope while highest order derivatives are zero:

\[\begin{split}\frac{d^n f(x)}{dx^n} = \begin{cases} a & \text{if } n = 1, \\ 0 & \text{if } n > 1 \end{cases}\end{split}\]

for all \(x \in \mathbb{R}\).

Parameters

  • x (array-like) – Values for which the derivative should be evaluated.

  • order (int, optional) – Order of the derivative. The default is 1.

Returns

Evaluated derivative. Array with a float dtype and the same shape as the broadcast of x with the function parameters.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the function parameters. If order is not a positive integer.

integrate(lower, upper)[source]

Compute the definite integral.

Note

The definite integral of a linear function is given by

\[\int_{x_1}^{x_2} f(x) dx = \frac{1}{2} a (x_2^2 - x_1^2) + b(x_2 - x_1)\]

for all \(x_1, x_2 \in \mathbb{R}\).

Parameters

  • lower (array-like) – Lower bound of the definite integral interval.

  • upper (array-like) – Upper bound of the definite integral interval.

Returns

Evaluated integral. Array with a float dtype and the same shape as the broadcast of lower and upper with the function parameters.

Return type

scalar or array

Raises

ValueError – If lower and upper cannot be broadcast with the function parameters.

Lognormal

class actuarial.distribution.Lognormal[source]

Bases: actuarial.superclass.Parametric.Parametric, actuarial.distribution.UnivariateDistribution.UnivariateDistribution

The log-normal distribution is a continuous probability distribution whose logarithm is normally distributed. Thus, if the random variable \(X\) is log-normally distributed, then \(Y = \ln(X)\) has a normal distribution. Equivalently, if \(Y\) has a normal distribution, then \(X = e^Y\), has a log-normal distribution. Therefore, the log-normal distribution takes only positive values and has the same parameters as a normal distribution denoted by \(\mu\) and \(\sigma\). Moreover, the log-normal distribution can be shifted by a real parameter \(A\) in order to be defined on a semi-infinite interval \([A,+\infty)\) instead. This distribution is a suitable choice to model the average cost of claims or individual claims with medium tail.

pdf(x)[source]

Evaluate the probability density function.

Note

The pdf of a shifted lognormal distribution is given by

\[f(x) = \frac{1}{(x-A)\sigma} \phi\left(\frac{\ln(x-A)-\mu}{\sigma}\right) = \frac{1}{(x-A) \sigma \sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{\ln(x-A) - \mu}{\sigma}\right)^2}\]

for \(x \geq A\).

Parameters

x (array-like) – Values for which the pdf should be evaluated.

Returns

Evaluated pdf. Array with a float dtype and the same shape as the broadcast of x with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the distribution parameters.

Note

The cdf of a shifted lognormal distribution is given by

\[F(x) = \Phi\left(\frac{\ln(x-A) - \mu}{\sigma}\right)\]

for \(x \geq A\).

quantile(p)[source]

Evaluate the quantile of the distribution.

Note

The quantile of a shifted lognormal distribution is given by

\[F^{-1}(p) = A + e^{\mu + \sigma\Phi^{-1}(p)}\]

for \(p \in (0,1)\).

Parameters

p (array-like) – Values (between 0 and 1) for which the quantile should be evaluated.

Returns

Evaluated quantile. Array with a float dtype and the same shape as the broadcast of p with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If p cannot be broadcast with the distribution parameters or contains values not between 0 and 1.

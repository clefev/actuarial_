Mixture

class actuarial.distribution.Mixture[source]

Bases: actuarial.superclass.Mixture.Mixture, actuarial.distribution.UnivariateDistribution.UnivariateDistribution

A mixture distribution is the probability distribution of a random variable that is derived from a collection of other random variables. It is constructed as follows: first, a random variable is selected by chance from the collection according to given probabilities of selection, and then the value of the selected random variable is realized.

The individual distributions \(X_k\) that are combined to form the mixture distribution are called the mixture components, and the probabilities (or weights) \(w_k \in (0,1)\) associated with each component are called the mixture weights. The mixture distribution is then equal to

\[\begin{split}X = \begin{cases} X_1 & \text{with probability } w_1, \\ ~ \vdots \\ X_K & \text{with probability } w_K \end{cases}\end{split}\]

where \(K\) is the number of mixture components and

\[\sum_{k=1}^K w_k = 1.\]
__init__(components, weights=None)[source]
Parameters

  • components (list of UnivariateDistribution) – List containing the components of the mixture distribution. Each component represents an univariate distribution.

  • weights (list of array-like or None) – Weights (or probabilities) associated with the mixture components. The list must have the same length as the number of components. The weights are normalized to get a sum equal to 1. The default assumes equiprobability for each mixture component.

Raises

  • TypeError – If components is not a list of UnivariateDistribution instances.

  • ValueError – If the length of weights is different from the length of components. If the elements of components and weights cannot be broadcast together. If negative weights are provided.

pdf(x)[source]

Evaluate the probability density function.

Note

The pdf of a mixture distribution is equal to the weighted sum of the pdf of each individual distribution:

\[f(x) = \sum_{k=1}^K w_k f_{X_k}(x)\]

for \(x \in \mathbb{R}\).

Parameters

x (array-like) – Values for which the pdf should be evaluated.

Returns

Evaluated pdf. Array with a float dtype and the same shape as the broadcast of x with the individual distributions and weights.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the individual distributions and weights.

quantile(p, maxiter=100, tol=0.0001)[source]

Evaluate the quantile of the distribution.

Note

No close formula exists to compute the quantile of a mixture distribution. Therefore, a bisection method is used in order to approximate it. The bisection finds the value \(x^*\) such that the cdf of the mixture distribution \(F(x^*)\) is equal to the provided probability \(p\). This value \(x^*\) is then returned as the quantile.

Parameters

  • p (array-like) – Values (between 0 and 1) for which the quantile should be evaluated.

  • maxiter (int, optional) – Maximum number of iterations for the bisection algorithm. The default is 100.

  • tol (float, optional) – Tolerance of the bisection algorithm. The maximum error between the computed quantile and the exact quantile will be equal to this tolerance. The default is 1e-4.

Returns

Evaluated quantile. Array with the same shape as the broadcast of p with the individual distributions and weights.

Return type

scalar or array

Raises

ValueError – If p cannot be broadcast with the individual distributions and weights, or contains values not between 0 and 1.

random(size=None, seed=- 1)[source]

Draw random samples of the distribution.

Note

A random sample of a mixture distribution is drawn as follows:

  • Select the individual distribution to generate by randomly drawing an integer \(k \in \lbrace1,...,K\rbrace\) where the probability to draw \(k\) is given by \(w_k\).

  • Draw a random sample for the selected k-th distribution \(X_k\).

Parameters

  • size (int or tuple of int or None, optional) – Number of samples to be drawn. If None, the instance shape will be used. The default is None.

  • seed (int or array-like of int or SeedSequence or Generator or None, optional) – Seed to use for the random number generator. The default uses the random number generator associated with this instance.

Returns

Random samples. Array with the same shape as size.

Return type

scalar or array

Raises

ValueError – If size is not a supershape of the instance shape.

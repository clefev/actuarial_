Pareto

class actuarial.distribution.Pareto[source]

Bases: actuarial.superclass.Parametric.Parametric, actuarial.distribution.UnivariateDistribution.UnivariateDistribution

The Pareto distribution is a continuous probability distribution defined on a semi-infinite interval \([A,+\infty)\) and parametrized by a positive shape parameter \(\alpha\). This distribution is characterized by a heavy tail. It is a particular case of the generalized Pareto distribution which has one additional parameter for more flexibility. It is therefore a suitable choice to model the cost of individual large claims for which the flexibility of the generalized Pareto distribution is not necessary.

pdf(x)[source]

Evaluate the probability density function.

Note

The pdf of a Pareto distribution is given by

\[f(x) = \frac{\alpha A^{\alpha}}{x^{\alpha + 1}}\]

for \(x \geq A\).

Parameters

x (array-like) – Values for which the pdf should be evaluated.

Returns

Evaluated pdf. Array with a float dtype and the same shape as the broadcast of x with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the distribution parameters.

cdf(x)[source]

Evaluate the cumulative distribution function.

Note

The cdf of a Pareto distribution is given by

\[F(x) = 1 - \left(\frac{x}{A}\right)^{-\alpha}\]

for \(x \geq A\).

x (array-like) – Values for which the cdf should be evaluated.

quantile(p)[source]

Evaluate the quantile of the distribution.

Note

The quantile of a Pareto distribution is given by

\[F^{-1}(p) = A(1 - p)^{-\alpha}\]

for \(p \in (0,1)\).

Parameters

p (array-like) – Values (between 0 and 1) for which the quantile should be evaluated.

Returns

Evaluated quantile. Array with a float dtype and the same shape as the broadcast of p with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If p cannot be broadcast with the distribution parameters or contains values not between 0 and 1.

Truncated

class actuarial.distribution.Truncated[source]

Bases: actuarial.distribution.UnivariateDistribution.UnivariateDistribution

A truncated distribution applies a conditional lower bound and/or upper bound to another distribution.

Let \(X\) denote the random variable to be truncated. We can build the random variable \(Y\) such that

\[Y = X \ | \ a < X \leq b\]

where \(a\) and \(b\) are respectively the lower bound and upper bound to apply to the distribution \(X\). The truncated distribution \(Y\) is therefore a conditional distribution that results from restricting the domain of another distribution \(X\). It is useful when the domain of a random variable is limited to values which lie above or below a given threshold or within a specified range.

pdf(x)[source]

Evaluate the probability density function.

Note

The pdf of a truncated distribution is given by

\[f(x | a < x \leq b) = \frac{f(x)}{F(b) - F(a)}\]

for \(x \in (a,b]\).

Parameters

x (array-like) – Values for which the pdf should be evaluated.

cdf(x)[source]

Evaluate the cumulative distribution function.

Note

The cdf of a truncated distribution is given by

\[F(x | a < x \leq b) = \frac{F(x) - F(a)}{F(b) - F(a)}\]

for \(x \in (a,b)\).

quantile(p)[source]

Evaluate the quantile of the distribution.

Note

The quantile of a truncated distribution is given by

\[F_{X | a < x \leq b}^{-1}(p) = F^{-1}\left(F(a) + (F(b)-F(a))p\right)\]

for \(p \in (0,1)\).

Parameters

p (array-like) – Values (between 0 and 1) for which the quantile should be evaluated.

Returns

Evaluated quantile. Array with the same dtype as the distribution to be truncated and the same shape as the broadcast of p with the distribution to be truncated, the lower bound and upper bound.

Return type

scalar or array

Raises

ValueError – If p cannot be broadcast with the distribution to be truncated, the lower bound and upper bound or contains values not between 0 and 1.

random(size=None, seed=- 1)[source]

Draw random samples of the distribution.

Note

A random sample of a truncated distribution is drawn by inverse cdf method as follows:

  • Draw a random sample \(u \in (0,1)\) from an uniform distribution.

  • Compute \(v = F(a) + \left(F(b)-F(a)\right)u\) such that \(v \in (F(a),F(b))\).

  • Return the inverse cdf \(F^{-1}(v)\).

Parameters

  • size (int or tuple of int or None, optional) – Number of samples to be drawn. If None, the instance shape will be used. The default is None.

  • seed (int or array-like of int or SeedSequence or Generator or None, optional) – Seed to use for the random number generator. The default uses the random number generator associated with this instance.

Returns

Random samples. Array with the same dtype as the distribution to be truncated and the same shape as size.

Return type

scalar or array

Raises

ValueError – If size is not a supershape of the instance shape.

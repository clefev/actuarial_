Generalized Pareto

class actuarial.distribution.GeneralizedPareto[source]

Bases: actuarial.superclass.Parametric.Parametric, actuarial.distribution.UnivariateDistribution.UnivariateDistribution

The generalized Pareto is a continuous probability distribution defined on a semi-infinite interval \([A,+\infty)\) and parametrized by two positive parameters, respectively the shape and scale parameters, denoted by \(\xi\) and \(\sigma\). This distribution is characterized by a heavy tail. It is therefore a suitable choice to model the cost of individual large claims or to model the tail of another distribution.

__init__(shape, scale, min=0)[source]
Parameters

  • shape (array-like) – Shape parameter \(\xi\).

  • scale (array-like) – Scale parameter \(\sigma\).

  • min (array-like, optional) – Minimum value \(A\). The default is 0.

Raises

ValueError – If shape and scale are not positive. If the parameters cannot be broadcast together.

pdf(x)[source]

Evaluate the probability density function.

Note

The pdf of a generalized Pareto distribution is given by

\[f(x) = \frac{1}{\sigma}\left( 1 + \xi \frac{x-A}{\sigma} \right)^{-\frac{1}{\xi} - 1}\]

where \(x \geq A\).

Parameters

x (array-like) – Values for which the pdf should be evaluated.

Returns

Evaluated pdf. Array with a float dtype and the same shape as the broadcast of x with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the distribution parameters.

cdf(x)[source]

Evaluate the cumulative distribution function.

Note

The cdf of a generalized Pareto distribution is given by

\[F(x) = 1 - \left( 1 + \xi \frac{x-A}{\sigma} \right)^{-1/\xi}\]

where \(x \geq A\).

Parameters

x (array-like) – Values for which the cdf should be evaluated.

Return type

quantile(p)[source]

Evaluate the quantile of the distribution.

Note

The quantile of a generalized Pareto distribution is given by

\[F^{-1}(p) = A + \frac{\sigma}{\xi} \left(\left( 1 - p \right)^{-\xi} - 1 \right)\]

where \(p \in (0,1)\).

Parameters

p (array-like) – Values (between 0 and 1) for which the quantile should be evaluated.

Returns

Evaluated quantile. Array with a float dtype and the same shape as the broadcast of p with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If p cannot be broadcast with the distribution parameters or contains values not between 0 and 1.

Univariate Distribution ABC

class actuarial.distribution.UnivariateDistribution[source]

Bases: abc.ABC, actuarial.superclass.Base.Base, actuarial.superclass.Broadcast.Broadcast, actuarial.superclass.Random.Random

Abstract class for constructing an univariate distribution.

An univariate distribution describes the probability distribution of a single random variable. The random variable can be discrete or continuous.

This abstract class implements basic methods such as

  • Evaluate the probability density/mass function (pdf).

  • Evaluate the cumulative distribution function (cdf).

  • Evaluate the quantile (quantile).

  • Draw random samples (random).

  • Compute the log-likelihood of a sample (log_likelihood).

Note that the type of values returned by the random variable (integers or floats) is provided by the dtype attribute of the instance.

log_likelihood(x, weights=None, axis=None, keepdims=False)[source]

Compute the log-likelihood.

Note

The log-likelihood of a sample \(x = (x_1, ..., x_n)\) is calculated as

\[L = \sum_{i=1}^n w_i \ln\left(f(x_i)\right)\]

where \(w = (w_1, ..., w_n)\) is a vector of weights associated with each \(x_i\).

Parameters

  • x (array-like) – Observed sample values from which the log-likelihood should be calculated.

  • weights (array-like or None, optional) – Weight associated with each value. The default assumes a weight of 1 associated with each observation.

  • axis (int or None, optional) – Axis along which the sum of the individual log-likelihood is computed. If None, the sum is computed over all the dimensions. The default is None.

  • keepdims (bool, optional) – If True, the axes which are reduced are left in the result as dimensions with size one (otherwise, they will be removed). With this option, the result will broadcast correctly against the input array x. The default is False.

Returns

Log-likelihood.

Return type

scalar or array

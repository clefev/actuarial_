Exponential

class actuarial.distribution.Exponential[source]

Bases: actuarial.superclass.Parametric.Parametric, actuarial.distribution.UnivariateDistribution.UnivariateDistribution

The shifted exponential distribution is a continuous probability distribution defined on a semi-infinite interval \([A,+\infty)\) and parametrized by a positive scale parameter denoted by \(\lambda\). The exponential distribution is a particular case of the gamma distribution. It is a suitable choice to model the average cost of claims or individual claims with thin tail.

__init__(scale=1, min=0)[source]
Parameters

  • scale (array-like, optional) – Scale parameter \(\lambda\). The default is 1.

  • min (array-like, optional) – Minimum value \(A\). The default is 0.

Raises

ValueError – If scale is not positive. If the parameters cannot be broadcast together.

pdf(x)[source]

Evaluate the probability density function.

Note

The pdf of a shifted exponential distribution is given by

\[f(x) = \frac{1}{\lambda} e^{-(x-A)/\lambda}\]

for \(x \geq A\).

Parameters

x (array-like) – Values for which the pdf should be evaluated.

Returns

Evaluated pdf. Array with a float dtype and the same shape as the broadcast of x with the distribution parameters.

Return type

scalar or array

Raises

quantile(p)[source]

Evaluate the quantile of the distribution.

Note

The quantile of a shifted exponential distribution is given by

\[F^{-1}(p) = A - \lambda \ln(1-p)\]

for \(p \in (0,1)\).

Parameters

p (array-like) – Values (between 0 and 1) for which the quantile should be evaluated.

Returns

Evaluated quantile. Array with a float dtype and the same shape as the broadcast of p with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If p cannot be broadcast with the distribution parameters or contains values not between 0 and 1.

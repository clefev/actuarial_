Bernoulli

class actuarial.distribution.Bernoulli[source]

Bases: actuarial.superclass.Parametric.Parametric, actuarial.distribution.UnivariateDistribution.UnivariateDistribution

The Bernoulli distribution is a discrete probability distribution of a random variable which takes the value 1 with probability \(p\) and the value 0 with probability \(1-p\):

\[\begin{split}X = \begin{cases} 0 & \text{probability } p, \\ 1 & \text{probability } 1-p. \end{cases}\end{split}\]

Less formally, it can be thought of as a distribution that models the possible outcomes of any single experiment that asks a yes-no question. Such questions lead to outcomes that are boolean-valued: a value 1 in case of success with probability \(p\) and a value 0 in case of failure with probability \(1-p\).

pdf(x)[source]

Evaluate the probability density function.

Note

The pdf of a Bernoulli distribution is given by

\[\begin{split}P(X = x) = \begin{cases} 1-p & \text{if } x = 0, \\ p & \text{if } x = 1. \end{cases}\end{split}\]
Parameters

x (array-like) – Values for which the pdf should be evaluated.

Returns

Evaluated pdf. Array with a float dtype and the same shape as the broadcast of x with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the distribution parameters.

cdf(x)[source]

Evaluate the cumulative distribution function.

Note

The cdf of a Bernoulli distribution is given by

\[\begin{split}F(x) = \begin{cases} 0 & \text{if } x < 0, \\ 1-p & \text{if } 0 \leq x < 1, \\ 1 & \text{if } x \geq 1. \end{cases}\end{split}\]

Normal

class distribution.Normal(mean=0, std=1, lower=- inf, upper=inf)

Normal distribution.

cdf(x)

Evaluate the cumulative distribution function.

Note

The cdf of a normal distribution is given by

\[F(x) = \Phi\left(\frac{x - \mu}{\sigma}\right)\]

where \(x \in \mathbb{R}\) and \(\Phi\) denotes the cdf the standard normal distribution given by

\[\Phi(x) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^x e^{-\frac{t^2}{2}} dt.\]
Parameters

x (array-like) – Values for which the cdf should be evaluated.

Returns

Evaluated cdf. Array with a float dtype and the same shape as the broadcast of x with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the distribution parameters.

pdf(x)

Evaluate the probability density function.

Note

The pdf of a normal distribution is given by

\[f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x - \mu}{\sigma}\right)^2}\]

where \(x \in \mathbb{R}\). In the case of a standard normal distribution (\(\mu = 0\) and \(\sigma = 1\)), the pdf can be written as

\[f(x) = \phi(x) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}.\]
Parameters

x (array-like) – Values for which the pdf should be evaluated.

Returns

Evaluated pdf. Array with a float dtype and the same shape as the broadcast of x with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the distribution parameters.

quantile(p)

Evaluate the quantiles of the distribution.

Parameters

p (array-like) – Values (between 0 and 1) for which the quantiles should be evaluated.

Returns

Evaluated quantiles. Array with a float dtype and the same shape as the broadcast of p with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If p cannot be broadcast with the distribution parameters.

random(size=None, seed=- 1)

Draw random samples of the distribution.

Parameters

  • size (int or tuple of int or None, optional) – Number of samples to be drawn. If None, the instance shape will be used. The default is None.

  • seed (int or array-like of int or SeedSequence or Generator or None, optional) – Seed to use for the random number generator. The default is to use the random number generator associated with this instance.

Returns

Random samples. Array with a float dtype and the same shape as size.

Return type

scalar or array

Raises

ValueError – If size is not a supershape of the instance shape.

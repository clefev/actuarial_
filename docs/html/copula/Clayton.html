cdf(x, keepdims=False)[source]

Evaluate the cumulative distribution function (cdf).

Note

The cdf of the Clayton copula is given by

\[C(u_1, ..., u_n) = \max\left(\sum_{i=1}^n u_i^{-\theta} + 1 - n, 0 \right)^{-1/\theta}\]

where \((u_1, ..., u_n) \in [0,1]^n\). In the bivariate case (\(n=2\)), we have therefore

\[C(u_1, u_2) = \max\left( u_1^{-\theta} + u_2^{-\theta} - 1, 0 \right)^{-1/\theta}.\]
Parameters

  • u (array-like) – Values (between 0 and 1) for which the cdf should be evaluated. The size along the last axis must be equal to the dimensionality n of the copula.

  • keepdims (bool, optional) – If True, the last axis which is reduced is left in the result as dimension with size one (otherwise, it will be removed). With this option, the result will broadcast correctly against the input array. The default is False.

Returns

Evaluated cdf. Array with the same shape as u except that the last axis has a size 1 if keepdims is True or is removed if it is False.

Return type

scalar or array

Raises

ValueError – If the size of u along its last axis is not equal to the dimensionality of the copula or if u contains values that are not between 0 and 1.

pdf(x, keepdims=False)[source]

Abstract method to compute the probability density function (pdf).

The first argument must be the values u for which the pdf should be evaluated. It should be an array-like of values between 0 and 1 and the size along the last axis must be equal to the dimensionality of the copula. A keepdims argument must also be provided in the method definition to remove or not the last axis which is reduced. Other additional arguments are allowed.

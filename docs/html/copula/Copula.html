abstract cdf(u, keepdims=False)[source]

Abstract method to compute the cumulative distribution function (cdf).

The first argument must be the values u for which the cdf should be evaluated. It should be an array-like of values between 0 and 1 and the size along the last axis must be equal to the dimensionality of the copula. A keepdims argument must also be provided in the method definition to remove or not the last axis which is reduced. Other additional arguments are allowed.

cdf()[source]

Decorator for the implemented cdf method of a Copula instance.

This decorator implements the following:

  • Convert the values u for which the cdf should be evaluated to a numpy array.

  • Control whether the size of u along its last axis is equal to the dimensionality of the copula.

  • Control whether the values of u are between 0 and 1.

  • Ensure that the returned output is an array with a float dtype and reduce it to a scalar when possible.

Returns

Gaussian

class actuarial.copula.Gaussian[source]

Bases: actuarial.superclass.Parametric.Parametric, actuarial.copula.Copula.Copula

The Gaussian copula is the copula that underlies the multivariate normal distribution.

The Gaussian copula is parametrized by a \(d \times d\) correlation matrix \(R\).

Moreover, the Gaussian copula can lead to particular copulas in the following way:

  • Independence copula if the correlation is set to 0 between all distinct variables (i.e. the correlation matrix is equal to the identity matrix).

  • Comonotonic copula if the correlation is set to 1 between all distinct variables (i.e. the correlation matrix contains ones everywhere).

  • Countermonotonic copula in the bivariate case where the correlation is set to -1 between the two variables.

cdf(u, keepdims=False)[source]

Evaluate the cumulative distribution function.

Note

The cdf of the Gaussian copula is given by

\[C(u_1, ..., u_d) = \Phi_R(\Phi^{-1}(u_1), ..., \Phi^{-1}(u_d))\]

where \((u_1, ..., u_d) \in [0,1]^d\) and \(\Phi_R\) is the joint cumulative distribution of a multivariate normal distribution with mean vector zero and covariance matrix equal to the correlation matrix \(R\).

Parameters

  • u (array-like) – Values (between 0 and 1) for which the cdf should be evaluated. The size along the last axis must be equal to the dimensionality of the copula.

  • keepdims (bool, optional) – If True, the last axis which is reduced is left in the result as dimension with size one (otherwise, it will be removed). With this option, the result will broadcast correctly against the input array. The default is False.

Returns

Evaluated cdf. Array with a float dtype and the same shape as u except that the last axis has a size 1 if keepdims is True or is removed if it is False.

Return type

scalar or array

Raises

ValueError – If the size of u along its last axis is not equal to the dimensionality of the copula or if u contains values that are not between 0 and 1.

pdf(u, keepdims=False)[source]

Evaluate the probability density function.

Note

The pdf of the Gaussian copula is given by

\[c(u_1, ..., u_d) = \dfrac{\phi_R(\Phi^{-1}(u_1), ..., \Phi^{-1}(u_d))}{\prod_{i=1}^d \phi(\Phi^{-1}(u_i))}\]

where \((u_1, ..., u_d) \in [0,1]^d\) and \(\phi_R\) is the joint density of a multivariate normal distribution with mean vector zero and covariance matrix equal to the correlation matrix \(R\).

Parameters

  • u (array-like) – Values (between 0 and 1) for which the pdf should be evaluated. The size along the last axis must be equal to the dimensionality of the copula.

  • keepdims (bool, optional) – If True, the last axis which is reduced is left in the result as dimension with size one (otherwise, it will be removed). With this option, the result will broadcast correctly against the input array. The default is False.

Returns

Evaluated pdf. Array with a float dtype and the same shape as u except that the last axis has a size 1 if keepdims is True or is removed if it is False.

Return type

scalar or array

Raises

ValueError – If the size of u along its last axis is not equal to the dimensionality of the copula or if u contains values that are not between 0 and 1.

random(size=None, seed=- 1)[source]

Draw d-dimensional random samples of joint uniform distributions.

Note

The following methodology is used to generate a d-dimensional random sample from the Gaussian copula:

  1. Draw a random sample \((x_1, ..., x_d)\) from a multivariate normal distribution with mean vector zero and covariance matrix equal to the correlation matrix \(R\).

  2. Return \((u_1, ..., u_d) = (\Phi(x_1), ..., \Phi(x_d))\) where \(\Phi\) denotes the cumulative distribution function of the standard normal distribution.

Parameters

  • size (int or tuple of int or None, optional) – Number of samples to be drawn. Because each sample is d-dimensional, the output shape will have an additional axis with size d in last position. If None, a single d-dimensional sample is returned. The default is None.

  • seed (int or array-like of int or SeedSequence or Generator or None, optional) – Seed to use for the random number generator. The default uses the random number generator associated with the instance.

Returns

Random samples. Array of values between 0 and 1 with the same shape as size except that an additional axis with size d has been added in last position.

array

The copula module provides all the tools needed to work with copulas.

Module description

The copula module can be easily imported from the actuarial package:

from actuarial import copula

Several classes are implemented in this module.

First, we can find an abstract class called Copula. This abstract class allows building copula models in a standardized way and with various features pre-implemented. Therefore, all copula models implemented in this package inherit from this abstract class. Moreover, it also allows you to build and customize your own copula models (by inheriting from this abstract class). Related documentation can be found below:

abstract cdf(u, keepdims=False)[source]

Abstract method to compute the cumulative distribution function (cdf).

The first argument must be the values u for which the cdf should be evaluated. It should be an array-like of values between 0 and 1 and the size along the last axis must be equal to the dimensionality of the copula. A keepdims argument must also be provided in the method definition to remove or not the last axis which is reduced. Other additional arguments are allowed.

cdf()[source]

Decorator for the implemented cdf method of a Copula instance.

This decorator implements the following:

  • Convert the values u for which the cdf should be evaluated to a numpy array.

  • Control whether the size of u along its last axis is equal to the dimensionality of the copula.

  • Control whether the values of u are between 0 and 1.

  • Ensure that the returned output is an array with a float dtype and reduce it to a scalar when possible.

Secondly, the module offers different copula models. For example, it implements the Gaussian copula, the comonotonic copula, the Clayton copula, … All these models inherit from the Copula abstract class as described above. In this way, they all share a common set of features (properties and methods) and they can be easily identified as a copula (using Python’s built-in functions isinstance or issubclass).

actuarial.decorator.catch_maturities(default1=0, default2=None)[source]

Decorator to catch the starting and ending maturities of a method.

The decorated function must be a method implemented within a class. The two first arguments of the method must be the starting and ending maturities, respectively maturities1 and maturities2. Other additional arguments are allowed.

This decorator implements the following:

  • Catch the maturities provided to the method assuming the following method definition: ([maturities1=default1,] maturities2=default2, **kwargs) i.e. the starting maturities maturities1 (first argument) is optional with default value to default1. It means that if only one argument is provided, it is assumed to be related to the ending maturities maturities2 while maturities1 will be set to its default value default1. If at least two arguments are provided, the first one will be considered as maturities1 and the second one will be maturities2. If no argument is provided, the starting and ending maturities will be set to their respective default values default1 and default2.

  • Convert maturities1 and maturities2 to numpy array if they are not set to None.

  • Control whether these maturities can be broadcast together or with the instance in case of Broadcast instance.

  • Shift maturities2 compared to maturities1 if a keyword argument shift is provided and set to True.

Parameters

  • default1 (array-like or None, optional) – Default value for the starting maturities. The default is 0.

  • default2 (array-like or None, optional) – Default value for the ending maturities. The default is None.

Returns

Decorator.

Return type

callable

Raises

ValueError – If the starting and ending maturities cannot be broadcast together or with the instance in case of Broadcast instance.

actuarial.decorator.fill_masked_array(fill_value=None)[source]

Decorator to replace the masked values of an output masked array. If the output is not a masked array, the output will be returned as is. If the output is a masked array, the masked values will be replaced by the provided fill_value argument in order to get a standard numpy array as output.

Parameters

fill_value (scalar or None, optional) – Value replacing the masked values of an output masked array: a standard numpy array will be returned where all masked values will be replaced by the provided fill_value. If None, the output masked array will be returned as is if it contains any masked values and if it doesn’t contain any masked values, a standard numpy array will be returned. The default is None.

Decorator.

Linear interpolation

class actuarial.yield_curve.LinearInterpolation[source]

Bases: actuarial.superclass.Piecewise.Piecewise, actuarial.superclass.TermStructure.TermStructure, actuarial.yield_curve.YieldCurve.YieldCurve

A linearly interpolated yield curve assumes that the value of the yield curve is linearly interpolated between two consecutive maturities.

Given a set of rates to interpolate \(\lbrace(\tau_i,r_i)\rbrace_{i=1,...,n}\), the linearly interpolated yield curve for a maturity \(\tau \in [\tau_i, \tau_{i+1}]\) is given by

\[r(\tau) = \alpha(\tau) r_i + (1-\alpha(\tau)) r_{i+1}\]

where \(\alpha(\tau) = \frac{\tau_{i+1}-\tau}{\tau_{i+1}-\tau_{i}} \in [0,1]\).

Note that below the minimum maturity \(\tau_1\) and above the maximum maturity \(\tau_n\), a flat extrapolation is used.

Note

Let \(\lbrace(\tau_i,r_i)\rbrace_{i=1,...,n}\) be a set of rates to interpolate.

The lineraly interpolated yield curve at maturity \(\tau\) is given by

\[\begin{split}r(\tau) = \begin{cases} r_1 & \text{if } \tau < \tau_1, \\ \alpha(\tau) r_i + (1-\alpha(\tau)) r_{i+1} & \text{if } \tau_i \leq \tau < \tau_{i+1}, \\ r_n & \text{if } \tau \geq \tau_n \end{cases}\end{split}\]

where \(\alpha(\tau) = \frac{\tau_{i+1}-\tau}{\tau_{i+1}-\tau_{i}} \in [0,1]\).

Note that below the minimum maturity \(\tau_1\) and above the maximum maturity \(\tau_n\), a flat extrapolation is used.

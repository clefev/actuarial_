Discounting Toolkit

class actuarial.yield_curve.Discounting[source]

Bases: actuarial.superclass.Arithmetic.Arithmetic, actuarial.superclass.Base.Base, actuarial.superclass.Broadcast.Broadcast, actuarial.superclass.Comparison.Comparison

Class to manage discount values between two maturities.

A discount value is a value that can be used to compute discount factors between two maturities, namely maturities1 and maturities2. There are different types of discount values such as discount factors, capitalization factors, compound rates, continuous rates and simple rates. The type of discount values is provided by the kind attribute. A discount value can be easily converted to one type to another using the convert method.

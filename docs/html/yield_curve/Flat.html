Flat

class actuarial.yield_curve.Flat[source]

Bases: actuarial.superclass.Parametric.Parametric, actuarial.yield_curve.YieldCurve.YieldCurve

A flat yield curve is characterized by a constant rate whatever the maturity.

The value of a flat yield curve for a maturity \(\tau\) is given by

\[r(\tau) = r_0\]

where \(r_0 \in \mathbb{R}\) is the constant rate. This constant rate represents the level of the yield curve. It can be compounded, continuously compounded or simple.

Parameters

Evaluate the value of the yield curve for the given maturities.

Note

The value of a flat yield curve at maturity \(\tau\) is given by

\[r(\tau) = r_0.\]

It is therefore constant and equal to the level parameter \(r_0\).

Yield Curve ABC

class actuarial.yield_curve.YieldCurve[source]

Bases: abc.ABC, actuarial.superclass.Base.Base, actuarial.superclass.Broadcast.Broadcast

Abstract class for constructing a yield curve.

A yield curve represents a relationship between maturities and interest rates. Interest rates can be compounded, continuously compounded or simple. Therefore, a yield curve has a kind attribute to describe the type of interest rates it provides. Note that a yield curve can also represent a relationship between maturities and discout factors (or capitalization factors) instead of interest rates.

This abstract class implements basic methods such as

  • Evaluate the value of the yield curve for given maturities (evaluate).

  • Compute the discount factor or capitalization factor for given maturities (discount_factor and capitalization_factor).

  • Convert the type of yield curve (convert).

  • Get a Discounting instance for given maturities (to_discounting).

Nelson-Siegel

class actuarial.yield_curve.NelsonSiegel[source]

Bases: actuarial.superclass.Parametric.Parametric, actuarial.yield_curve.YieldCurve.YieldCurve

The Nelson-Siegel model is a parametric model for constructing yield curves.

The Nelson-Siegel model has 4 parameters, it models the value of the yield curve at matrurity \(\tau\) as follows:

\[r(\tau) = \beta_0 + \beta_1 \frac{1 - e^{-\frac{\tau}{\lambda}}}{\frac{\tau}{\lambda}} + \beta_2 \left( \frac{1 - e^{-\frac{\tau}{\lambda}}}{\frac{\tau}{\lambda}} - e^{-\frac{\tau}{\lambda}} \right)\]

where \(\beta_0\) is the level parameter, \(\beta_1\) the slope parameter, \(\beta_2\) the curvature parameter and \(\lambda\) a non-negative scale parameter.

Note that the Nelson-Siegel model can be seen as a linear model in beta coefficients. By convention, the Nelson-Siegel model is used to model spot rates which are continuously compounded.

evaluate(maturities, forward=False, split=False)[source]

Evaluate the value of the yield curve for the given maturities.

Note

The Nelson-Siegel formula for the spot rate at maturity \(\tau\) is given by

\[r(\tau) = \beta_0 + \beta_1 \frac{1 - e^{-\frac{\tau}{\lambda}}}{\frac{\tau}{\lambda}} + \beta_2 \left( \frac{1 - e^{-\frac{\tau}{\lambda}}}{\frac{\tau}{\lambda}} - e^{-\frac{\tau}{\lambda}} \right),\]

and the instantaneaous forward rate is given by

\[f(\tau) = \beta_0 + \beta_1 e^{-\frac{\tau}{\lambda}} + \beta_2 \frac{\tau}{\lambda} e^{-\frac{\tau}{\lambda}}.\]
Parameters

  • maturities (array-like) – Maturities for which the yield curve should be evaluated.

  • forward (bool, optional) – If True, the instantaneaous forward rate is returned. Otherwise, the spot rate is returned. The default is False.

  • split (bool, optional) – If True, the three additive components of the Nelson-Siegel model (level, slope and curvature components) are returned separately through a tuple. Otherwise, they are aggregated. The default is False.

Returns

Value of the yield curve for the given maturities. If split is True, a tuple with the three additive components of the Nelson-Siegel model is returned.

Return type

scalar or array or tuple

Raises

ValueError – If maturities cannot be broadcast with the instance.

predictors(maturities, scale=None, forward=False)[source]

Compute the predictors of the Nelson-Siegel model.

Note

The Nelson-Siegel model can be seen as a linear model with beta coefficients.

The spot rate can be written as

\[r(\tau) = \beta_0 r_0(\tau; \lambda) + \beta_1 r_1(\tau; \lambda) + \beta_2 r_2(\tau; \lambda)\]

where the predictors \(r_i(\tau; \lambda)\) depend on the maturity \(\tau\), scale parameter \(\lambda\) and are given by

\[\begin{split}\begin{cases} r_0(\tau; \lambda) = 1 \\ r_1(\tau; \lambda) = \frac{1 - e^{-\frac{\tau}{\lambda}}}{\frac{\tau}{\lambda}} \\ r_2(\tau; \lambda) = \frac{1 - e^{-\frac{\tau}{\lambda}}}{\frac{\tau}{\lambda}} - e^{-\frac{\tau}{\lambda}} \end{cases}\end{split}\]

The instantaneous forward rate can be written as

\[f(\tau) = \beta_0 f_0(\tau; \lambda) + \beta_1 f_1(\tau; \lambda) + \beta_2 f_2(\tau; \lambda)\]

where the predictors \(f_i(\tau,\lambda)\) are given by

\[\begin{split}\begin{cases} f_0(\tau; \lambda) = 1 \\ f_1(\tau; \lambda) = e^{-\frac{\tau}{\lambda}} \\ f_2(\tau; \lambda) = \frac{\tau}{\lambda} e^{-\frac{\tau}{\lambda}} \end{cases}\end{split}\]
Parameters

  • maturities (array-like) – Maturities for which the yield curve should be evaluated.

  • scale (array-like or None, optional) – Scale parameter of the Nelson-Siegel model. The default uses the scale parameter of the instance.

  • forward (bool, optional) – If True, the predictors of the instantaneaous forward rate are returned. Otherwise, the predictors of the spot rate are returned. The default is False.

Returns

The three predictors of the Nelson-Siegel model. Each predictor is an array with a shape corresponding to the broadcast of the maturities and scale parameter.

Return type

tuple of array

Raises

ValueError – If maturities and scale cannot be broadcast together.

classmethod fit(yield_curve, scales)[source]

Fit a Nelson-Siegel model.

Note

The fitting method uses a grid search method for the scale parameter and an ordinary least-squares (OLS) method for the beta parameters.

Given a scale parameter \(\lambda\), the beta parameters of the Nelson-Siegel model are fitted by ordinary least-squares method. It means that the beta parameters \(\beta_0, \beta_1\) and \(\beta_2\) are obtained by minimizing the sum of squared error (SSE) given by

\[SSE = \sum_j \left( \beta_0 + \beta_1 x_1(\tau_j;\lambda) + \beta_2 x_2(\tau_j;\lambda) - r^{obs}(\tau_j) \right)^2\]

where \(r^{obs}(\tau_j)\) are the observed spot continuous rates to be fitted.

In order to fit the scale parameter, a grid serach method is used. It means that a grid of scale parameters \((\lambda_1, ..., \lambda_K)\) must be provided to the algorithm. For each scale parameter \(\lambda_k\), the beta parameters are fitted by OLS method and the sum of squared error \(SSE_k\) is calculated. The optimal scale parameter is the one for which the SSE is the smallest.

Parameters

  • yield_curve (RawYieldCurve) – Raw yield curve to be fitted. The curve will be automatically converted to spot continuous rates for the fitting.

  • scales (1D array-like) – Grid search to use for the scale parameter. If only one number is provided, the scale parameter will be forced to this one. Otherwise, the optimal scale parameter will be chosen from the grid.

Returns

Fitted instance of a Nelson-Siegel model.

Return type

NelsonSiegel

Raises

  • TypeError – If yield_curve is not a RawYieldCurve instance.

  • ValueError – If the length of yield_curve is not at least 3.

class actuarial.parameter.Parameters[source]

Bases: actuarial.superclass.Dictionary.Dictionary, actuarial.superclass.Base.Base, actuarial.superclass.Broadcast.Broadcast

Class representing a collection of parameters.

This class inherits from the Dictionary superclass. Therefore, it represents a collection of key-parameter pairs. The key is a string representing the name of the parameter and the parameter is a Parameter instance. This class allows to control, update and reset several parameters at once.

  • *args (array-like or Parameter or None) – New parameter values. The first argument is the new value of the first parameter, the second argument is the new value of the second parameter, … The number of arguments provided must be equal to the number of parameters of the instance.

  • **kwargs (array-like or Parameter or None) – New parameter values given as key-value pairs. The keyword is the key of the parameter to update. If the key doesn’t exist, an error will be raised.

control()[source]

Control whether the value of each parameter is valid or not.

For each parameter, the associated control method will be called. If a parameter is not valid, an error will be raised. If the instance contains a parameter called min and a parameter called max, an additional control is performed to ensure that min < max.

Raises

ValueError – If a parameter is not valid. If the instance contains a parameter called min and a parameter called max for which min >= max.

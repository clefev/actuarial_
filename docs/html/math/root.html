Root-finding

actuarial.math.bisection(f, lower, upper, maxiter=100, tol=0.0001, f_tol=0.0, kwargs={})[source]

Bisection algorithm to find the root of a continuous function in an interval \([a,b]\).

The algorithm supports vectorization. It means that if the function returns n values, then n roots will be found. Note that the algorithm will stop as soon as one of the criteria given by tol or f_tol is satisfied (not when both are satisfied at the same time).

Parameters

  • f (callable) – Function whose root must be found. The function must be continuous and return a scalar or an array. Moreover, \(f(a)\) and \(f(b)\) must have opposite signs. If an array is returned, a root will be found for each returned component of the function (vectorization). The function will be called as f(x, **kwargs) where x is the value for which the function should be evaluated and kwargs is a dictionary of additional arguments that can be passed to the function (see below).

  • lower (array_like) – Lower bound \(a\) of the interval in which the root is.

  • upper (array_like) – Upper bound \(b\) of the interval in which the root is.

  • maxiter (int, optional) – Maximum number of iterations. The algorithm will stop and return the current result after this number of iterations. The default is 100.

  • tol (float, optional) – Absolute tolerance for the root \(x^*\). The maximum absolute error between the computed root \(x^*\) and the exact root \(x_{exact}\) will be equal to this tolerance. The default is 1e-4.

  • f_tol (float, optional) – Absolute tolerance for \(f(x^*)\). The maximum absolute value for \(f(x^*)\) will be equal to this tolerance. The default is 0.

  • kwargs (dict, optional) – Additional keyword arguments for the function f.

Raises

ValueError – If upper is not greater than lower. If f(lower) and f(upper) don’t have opposite signs.

Returns

  • root (scalar or array) – Computed root.

  • f_root (scalar or array) – Value of the function for the computed root.

  • iter (int) – Number of iterations needed to find the root.

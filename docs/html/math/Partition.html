index(x)[source]

Return the partition index associated with input values.

Note

The partition index associated with a value \(x\) is the index \(i\) satisfying either

\[x_i \leq x < x_{i+1}\]

in case of left-closed interval or

\[x_i < x \leq x_{i+1}\]

in case of right-closed interval.

Note that if the value \(x\) is below \(x_0\), the returned index will be truncated to \(i=0\). And if it is above \(x_n\), the returned index will be \(i=n\). Therefore, the returned index is always an integer between 0 and \(n\).

Mixture

class actuarial.yield_curve.Mixture[source]

Bases: actuarial.superclass.Mixture.Mixture, actuarial.yield_curve.YieldCurve.YieldCurve

A mixture yield curve is a yield curve derived from a collection of other yield curves. The other yield curves are called the components of the mixture yield curve. For each component, a weight can be associated.

The value of a mixture yield curve is expressed as a linear combination of its components:

\[r(\tau) = \sum_{k=1}^K w_k r_k(\tau)\]

where \(r_k(\tau)\) is the value of the k-th yield curve component at maturity \(\tau\), \(w_k\) the associated weight and \(K\) the number of mixture components.

__init__(components, weights=None, kind='compound_rate')[source]
Parameters

  • components (list of YieldCurve) – List containing the yield curve components.

  • weights (list of array-like or None, optional) – Weights associated with the mixture components. The list must have the same length as the number of components. The default assumes a weight of 1 associated with each component.

  • kind ({"compound_rate", "continuous_rate", "simple_rate"}, optional) – Type of yield curve. The default assumes compound rates.

Raises

  • TypeError – If components is not a list of YieldCurve elements.

  • ValueError – If kind is not valid. If the length of weights is not the same as the length of components. If the weights and components cannot be broadcast together.

evaluate(maturities, split=False)[source]

Evaluate the value of the yield curve for the given maturities.

Note

The value of a mixture yield curve at maturity \(\tau\) is given by

\[r(\tau) = \sum_{k=1}^K w_k r_k(\tau)\]

where \(r_k(\tau)\) is the value of the k-th yield curve component at maturity \(\tau\) and \(w_k\) the associated weight.

Note that the values of each component of the mixture yield curve are automatically converted to the same type as the mixture yield curve (kind attribute of the instance).

Parameters

  • maturities (array-like) – Maturities for which the yield curve should be evaluated.

  • split (bool, optional) – If True, the individual weighted components of the mixture yield curve are returned separately. Otherwise, they are aggregated. The default is False.

Returns

Value of the yield curve for the given maturities. If split is True, a list with the weighted values of each individual component is returned.

Return type

scalar or array or list

Raises

ValueError – If maturities cannot be broadcast with the instance.

Comparison

class actuarial.superclass.Comparison[source]

Bases: object

Superclass to handle comparison operators such as equal to, not equal to, less than, greater than, less than or equal to and greater than or equal to.

All these comparison operators are available at once by implementing the __comparison__ abstract method in the derived class. This abstract method describes how comparison operators (==, !=, <, >, <= and >=) should behave on the object.

abstract __comparison__(value, operator)[source]

Abstract method describing how comparison operators (==, !=, <, >, <= and >=) should behave on the object.

The operator argument is a function representing a generic comparison operator.

For example, the derived class could implement the following __comparison__ method:

def __comparison__(self, value, operator):
    return operator(self.attr, value)

By implementing this single abstract method, all comparison operators mentioned above will be available (e.g., object == value will return the result of object.attr == value, object != value will return the result of object.attr != value, …).

Import the Comparison superclass.

from actuarial.superclass import Comparison

Create a class Example inheriting from the Comparison superclass and having an amount attribute representing a number. The application of comparison operators on the instance (==, !=, <, >, <= and >=) should return the comparison with the amount attribute of the instance.

class Example(Comparison):

    def __init__(self, amount: float):
        self.amount = float(amount)

    def __comparison__(self, value, operator):
        return operator(self.amount, value) # return the comparison with the 'amount' attribute of the instance

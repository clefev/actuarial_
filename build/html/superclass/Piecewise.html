Piecewise

class actuarial.superclass.Piecewise[source]

Bases: object

Superclass for building piecewise models.

A piecewise model is a model having a piecewise function. A linearly interpolated yield curve is an example of piecewise model. The interpolation is indeed done through a linear piecewise function.

Each Piecewise instance shares the following features:

  • The piecewise function is accessible through the piecewise_function attribute.

  • A broadcast method exists to mimic the broadcast of the piecewise function with input arguments.

abstract property piecewise_function

Abstract property giving the piecewise function used by the class. The property must return a PiecewiseFunction instance and must be decorated by the property and actuarial.decorator.memorize decorators.

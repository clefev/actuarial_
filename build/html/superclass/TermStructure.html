TermStructure

class actuarial.superclass.TermStructure[source]

Bases: object

Superclass for building discrete term structure models.

A discrete term structure model is described by a finite set of values associated with maturities.

A (discrete) yield curve is an example of term structure model. It represents a relationship between interest rates and maturities. Mathematically, it can be described by a collection \(\lbrace (\tau_i, r_i) \rbrace_{i=1,...,n}\) where \(\tau_i\)’s are the maturities, \(r_i\)’s are the associated interest rates and \(n\) is the finite number of observed pairs.

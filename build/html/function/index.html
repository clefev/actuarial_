class actuarial.function.Function[source]

Bases: abc.ABC, actuarial.superclass.Base.Base, actuarial.superclass.Broadcast.Broadcast

Abstract class for constructing a real function.

A real function is a function that takes as input a real number, commonly represented by the variable \(x\), for producing another real number, the value of the function, commonly denoted \(f(x)\). Its domain is therefore the set (or a subset) of real numbers \(\mathbb{R}\):

\[f : X \longrightarrow \mathbb{R}\]

where \(X \subseteq \mathbb{R}\) is the domain of the function.

This abstract class implements basic methods such as

  • Evaluate the value of the function (evaluate and __call__).

  • Compute derivatives (derivative).

  • Compute definite integrals (integrate).

Abstract methods

abstract evaluate(x)[source]

Abstract method to evaluate the value of the function.

The first argument x must be the values for which the function should be evaluated. Other additional arguments are allowed.

abstract derivative(x, order=1)[source]

Abstract method to compute the derivative.

The first argument x must be the values for which the derivative should be evaluated. The second argument order must be a positive integer representing the order of the derivative (the default assumes a first order derivative). Other additional arguments are allowed.

abstract integrate(lower, upper)[source]

Abstract method to compute the definite integral.

The two first arguments lower and upper must be respectively the lower bound and upper bound of the definite integral. Other additional arguments are allowed.

evaluate()[source]

Decorator for the implemented evaluate method of a Function instance.

This decorator implements the following:

  • Convert the values for which the function should be evaluated to a numpy array.

  • Control whether these values can be broadcast with the instance.

  • Ensure that the returned output is an array and reduce it to a scalar when possible.

derivative()[source]

Decorator for the implemented derivative method of a Function instance.

This decorator implements the following:

  • Convert the values for which the derivative should be evaluated to a numpy array.

  • Control whether these values can be broadcast with the instance.

  • Control whether the order of the derivative is a positive integer.

  • Ensure that the returned output is an array with a float dtype and reduce it to a scalar when possible.

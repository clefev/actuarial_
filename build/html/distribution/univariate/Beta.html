Beta

class actuarial.distribution.Beta[source]

Bases: actuarial.superclass.Parametric.Parametric, actuarial.distribution.UnivariateDistribution.UnivariateDistribution

The beta distribution is a continuous probability distribution defined on a finite interval \([a,b]\) (typically, the unit interval \([0,1]\)) and parameterized by two positive parameters, denoted by \(\alpha\) and \(\beta\), that control the shape of the distribution. Therefore, the beta distribution is a suitable model for the random behavior of percentages and proportions.

Parameters

  • shape1 (array-like) – First shape parameter \(\alpha\).

  • shape2 (array-like) – Second shape parameter \(\beta\).

  • max (array-like, optional) – Maximum value \(b\). The default is 1.

Raises

ValueError – If shape1 and shape2 are not positive. If max is not greater than min. If the parameters cannot be broadcast together.

pdf(x)[source]

Evaluate the probability density function.

Note

The pdf of a four-parameters beta distribution is given by

\[f(x) = \frac{(x-a)^{\alpha-1} (b-x)^{\beta-1}}{(b-a)^{\alpha+\beta-1} B(\alpha,\beta)}\]

for \(x \in [a,b]\).

In the case of a two-parameters beta distribution (\(a=0\) and \(b=1\)), the pdf can be simplified as

\[f(x) = \frac{x^{\alpha-1} (1-x)^{\beta-1}}{B(\alpha,\beta)}\]

for \(x \in [0,1]\).

Parameters

x (array-like) – Values for which the pdf should be evaluated.

Returns

Evaluated pdf. Array with a float dtype and the same shape as the broadcast of x with the distribution parameters.

Return type

scalar or array

Raises

ValueError – If x cannot be broadcast with the distribution parameters.

cdf(x)[source]

Evaluate the cumulative distribution function.

Note

The cdf of a four-parameters beta distribution is given by

\[F(x) = \frac{B\left(\frac{x-a}{b-a};\alpha,\beta\right)}{B(\alpha,\beta)}\]

for \(x \in [a,b]\).

In the case of a two-parameters beta distribution (\(a=0\) and \(b=1\)), the cdf can be simplified as

\[F(x) = \frac{B\left(x;\alpha,\beta\right)}{B(\alpha,\beta)}\]

for \(x \in [0,1]\).

Parameters

x (array-like) – Values for which the cdf should be evaluated.

Returns

Evaluated cdf. Array with a float dtype and the same shape as the broadcast of x with the distribution parameters.

Return type

scalar or array

Raises

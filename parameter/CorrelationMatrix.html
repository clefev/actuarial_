Correlation matrix parameter.

A correlation matrix parameter is a parameter for which its value is a correlation matrix (symmetric positive semi-definite matrix with diagonal 1 and values between -1 and 1).

value

Value of the parameter. The parameter is not initialized if its value is set to None.

Type

array or None

set(value)

Set the value of the parameter.

Parameters

value (array-like or Parameter or None) – The new value of the parameter.

Returns

Instance.

Return type

Parameter

control(value=None, name='value')[source]

Control whether the value of the parameter is valid or not.

The value of the parameter is valid if it satisfies all the following conditions (i.e. if it can be considered as a correlation matrix):

  • Matrix (array with 2 dimensions).

  • Square matrix (having the same number of rows and columns).

  • Symmetric matrix (equal to its transpose).

  • Values between -1 and 1 (correlation coefficients).

  • Diagonal containing only 1.

  • Positive semi-definite (all eigen values must be non-negative).

Parameters

  • value (array-like or None, optional) – The value to control. If None, the value of the instance will be used instead. The default is None.

  • name (str, optional) – Name of the parameter. If an error is raised, this name will be used. The default is “value”.

Raises

ValueError – If the parameter is not initialized (set to None). If one of the condition above is not fulfilled.

reset()

Reset the value of the parameter.

The value of the parameter will be set to None as if it had not been initialized.
